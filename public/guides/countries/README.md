## Guide Structure

Each guide follows a standardized structure:

1. **Executive Summary** - Key rates, effective tax rates, expat highlights
2. **Income Tax Structure** - Brackets, rates, calculation methodology
3. **Social Contributions** - Pension, health, unemployment insurance
4. **Expat-Specific Regimes** - Special tax schemes for foreign workers
5. **Deductions & Credits** - Standard deductions, allowances, credits
6. **Regional Variations** - State/provincial/municipal differences
7. **Tax Residency Rules** - 183-day rule, permanent home test, etc.
8. **Calculation Examples** - Step-by-step examples at various income levels
9. **Official Sources** - Government agencies, tax authorities, professional firms
10. **Implementation Notes** - YAML config guidance, test vectors, edge cases

## Usage

These guides are intended to:
- Help users understand tax systems before relocating
- Provide source material for creating tax configurations
- Document research methodology for tax calculations
- Serve as reference for calculator validation
